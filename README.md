# weather-app-challenge
## Requirements

## User Story
  As a keen traveller
	I want to know the weeks weather forecast
	for my travel destination
	so that I can pack appropriately
## Acceptance Criteria
  I can see a summary of the weeks weather forecast for my desired location.
